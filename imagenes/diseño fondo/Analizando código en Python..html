    El uso de self es una convención en Python, y aunque el nombre self no es obligatorio (podrías usar cualquier otro nombre), es altamente recomendado por razones de claridad y consistencia en el código.

    En Python, puedes declarar una variable global utilizando la palabra clave global. Esto le dice a Python que la variable está destinada a ser global en lugar de local dentro de una función. Aquí tienes un ejemplo:

    python
    # Declarar una variable global
mi_variable_global = 42

def mi_funcion():
    # Acceder a la variable global
    print(mi_variable_global)

    # Modificar la variable global
    global mi_variable_global
    mi_variable_global = 99

    En este ejemplo:

    Es importante tener cuidado al usar variables globales, ya que pueden hacer que el código sea menos modular y más difícil de entender. En general, se recomienda evitar el uso excesivo de variables globales y, en su lugar, pasar valores como argumentos a funciones cuando sea posible.
